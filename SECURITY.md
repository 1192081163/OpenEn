# Security Policy

## 支持范围

当前项目处于早期开发阶段。安全问题请优先基于最新的 `main` 分支确认。

## 报告安全问题

如果你发现安全问题，请不要直接公开可利用细节。可以先通过私下渠道联系维护者，或在 GitHub 上创建不包含复现密钥、真实账户信息和敏感网页数据的 issue。

报告中建议包含：

- 受影响的浏览器和版本
- 受影响的 OpenEn 版本或 commit
- 问题描述
- 最小复现步骤
- 预期影响

## 密钥和隐私

- OpenEn 不需要把 DeepSeek API Key 提交到仓库。
- API Key 由用户在扩展 UI 中输入，并保存在浏览器扩展本地存储。
- 不要把真实 API Key、浏览历史、网页私有内容或生词本导出文件附在公开 issue 中。

## 浏览器权限

OpenEn 需要读取网页选中文本、显示划词弹窗并高亮生词，因此浏览器会提示网页内容相关权限。请只在你信任的浏览器环境中启用扩展，并按需授予网站访问权限。
